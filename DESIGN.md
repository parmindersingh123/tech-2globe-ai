# Design Brief — Tech 2Globe AI (Solution Pages + Contact Us + AI Development Company)

## Overview
**Solution Pages:** Premium long-scroll service solution page extending homepage design. 12 visually distinct alternating sections, smooth scroll animations, glass-morphism effects, generous typography (56–64px hero, 36px sections), minimum 5000px height desktop.

**Contact Us Page:** Premium AI-focused contact experience, futuristic neural network aesthetic, glassmorphic form, floating nodes, circuit patterns. Hero CTA → Form → Social proof → FAQ → Trust CTA. Minimum 3500px height desktop. Fully responsive mobile-first. White background with dark text throughout.

**AI Development Company (`/ai-development-company`):** Futuristic dark space aesthetic showcase. Deep dark background (#0A0A0F), glassmorphism with neon glow borders, animated gradient flows, floating orbs. Bold 3-color gradient headlines (blue → purple → cyan). Premium tech showcase. Minimum 6000px height desktop.

## Tone & Aesthetic
**Solution Pages:** Futuristic premium tech, bold confident voice. Every section visually distinct—alternating backgrounds (full dark, dot grid, glow accent, card surface). Glass-morphism + glow borders signature detail. Scroll-triggered animations via Motion: fade-up, counter-up, slide-in, stagger. No heavy libraries or 3D effects. Anti-generic: electric blue paired with purple, high contrast typography, generous whitespace.

**Contact Us:** Premium futuristic AI, confident and trustworthy. Neural network nodes, circuit patterns, floating geometric elements. Glassmorphism, blue/purple/teal glow accents, high-contrast typography. White backgrounds, dark text throughout — NO dark theme. Smooth micro-interactions on focus/hover.

**AI Development Company:** Premium dark space aesthetic. Deep dark (#0A0A0F) background, glassmorphic cards with neon glow borders, animated gradient flows (8s loop), floating pulsing orbs. Bold 3-color gradient headlines (blue #4F8EF7 → purple #7C5CBF → cyan #06B6D4). Neon glow effects on hover. Pulsing node animations. High-contrast white text on dark. Smooth entrance animations. Signature detail: neon-outlined cards + floating glowing orbs + animated gradient background + glow-pulse on card hover.

## Constraints & Safeguards
- **No raw hex/named colors:** All colors via CSS variables and Tailwind theme
- **No dark theme:** Contact Us page always white background, dark text throughout
- **Dark theme scoped:** `/ai-development-company` only uses `.dark` class (deep background), reuses existing nav/footer
- **No arbitrary colors:** Never use `bg-[#123]` or inline `style=` attributes
- **Mobile-first:** sm:, md:, lg: breakpoints for all layouts
- **Contrast:** AA+ for all text on backgrounds, 4.5:1 for body text, 3:1 for large text
- **Font loading:** font-display: swap for instant render, no invisible text
- **Hover states:** Every interactive element must signal (scale, glow, color, underline)
- **Animations lightweight:** No 3D transforms, no heavy particle systems, GPU-optimized (transform/opacity only)
- **Image fallback:** Use SVG placeholders or brand icons if imagery missing
- **Form accessibility:** Labels associated via htmlFor, error announcements, focus indicators
- **Touch targets:** Button min 48×48px, form inputs 48px height
- **Dark mode CSS:** All dark theme via `.dark` class selector in existing utilities, no separate stylesheet

## Signature Detail
**Solution Pages:** Glowing accent borders + glass cards + dot grid alternation. Electric blue glow on interactive hover, glass-morphism on elevated cards, dot grid pattern for visual rhythm on alternating sections. Creates premium, futuristic depth without visual noise.

**Contact Us:** Floating neural network nodes + glassmorphic form + blue glow accents on focus. Pulsing nodes create depth and motion without distraction. Glass effect elevates form from background. Blue glow on form fields creates premium AI aesthetic. Gradient accent lines create visual continuity with brand.
